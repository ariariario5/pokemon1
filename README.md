# 初代ポケモン風バトルゲーム

初代ポケモン（赤・緑）のバトルシーンを再現したWebゲームです。

## 特徴

- **レトロなUI**: 初代ポケモンのバトル画面を忠実に再現
- **ターン制バトル**: プレイヤーと敵が交互に行動するクラシックなバトルシステム
- **アニメーション**: 攻撃時の振動エフェクトとダメージ表示
- **サウンドエフェクト**: Web Audio APIを使用した8bit風の効果音
- **完全レスポンシブ**: デスクトップとモバイルデバイスに対応

## 実装されている機能

### ポケモン
- **ヒトカゲ** (プレイヤー): ほのおタイプ
- **フシギダネ** (敵): くさタイプ

### 技
- **ひのこ** - ほのおタイプの攻撃技
- **たいあたり** - ノーマルタイプの基本攻撃
- **ひっかく** - ノーマルタイプの攻撃技
- **なきごえ** - 相手の防御力を下げる補助技

### バトルシステム
- ダメージ計算（攻撃力、防御力、タイプ相性を考慮）
- HPバーの色変化（緑→黄→赤）
- 勝敗判定
- ランダムな敵の行動

## 使い方

1. ブラウザでindex.htmlを開く
2. 「たたかう」を選択して技を選ぶ
3. 敵のHPを0にして勝利を目指す

## 操作方法

- **たたかう**: 攻撃技を選択
- **ポケモン**: 他のポケモンに交代（未実装）
- **どうぐ**: アイテムを使用（ダミー）
- **にげる**: バトルから逃げる

## ファイル構成

```
pokemon-battle/
├── index.html      # メインのHTMLファイル
├── style.css       # スタイルシート
├── script.js       # ゲームロジック
└── README.md       # このファイル
```

## ブラウザで実行

ローカルサーバーを起動する場合：

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsのlive-serverがある場合
npx live-server
```

その後 http://localhost:8000 にアクセス

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: Flexbox/Grid、アニメーション、レスポンシブデザイン
- **JavaScript ES6+**: モジュール化された関数、イベント処理
- **Web Audio API**: リアルタイムサウンド生成

## カスタマイズ

`script.js`の`pokemonData`オブジェクトを編集することで、新しいポケモンや技を追加できます：

```javascript
const pokemonData = {
    newPokemon: {
        name: "新しいポケモン",
        type: "でんき",
        hp: 25,
        attack: 15,
        defense: 10,
        moves: [
            { name: "10まんボルト", power: 12, type: "でんき", pp: 15 }
        ]
    }
};
```
